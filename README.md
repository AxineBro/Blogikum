
# Блогикум

Социальная сеть для публикации личных дневников (учебный проект).

## Текущая функциональность

### Реализованные страницы:
- **Главная страница** - отображает список всех публикаций
- **Страница поста** - показывает полный текст выбранной публикации  
- **Страница категории** - отображает категории из URL
- **Статические страницы** - "О проекте" и "Правила сайта"

### Маршруты:

#### Приложение `blog`:
- `/` - главная страница
- `/posts/<id>/` - детальная страница поста
- `/category/<category_slug>/` - страница категории

#### Приложение `pages`:
- `/pages/about/` - страница "О проекте"
- `/pages/rules/` - страница "Правила сайта"

## Технологии

- Python 3.10+
- Django 3.2+

## Установка и запуск

1. Активируйте виртуальное окружение:
```bash
source venv/bin/activate  # Linux/MacOS
venv\Scripts\activate     # Windows
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите сервер разработки:
```bash
cd blogicum
python manage.py runserver
```

4. Откройте в браузере: `http://127.0.0.1:8000/`

## Структура данных

Проект использует временные данные в виде списка словарей с постами:

```python
posts = [
    {
        'id': 0,
        'location': 'Остров отчаянья',
        'date': '30 сентября 1659 года', 
        'category': 'travel',
        'text': 'Текст поста...'
    },
    # ...
]
```

## Особенности реализации

- Использование Django-приложений `blog` и `pages`
- Namespace для URL: `blog:` и `pages:`
- Базовый шаблон с наследованием
- Обработка несуществующих постов (404)
- Статические файлы подключены на уровне проекта
